# Security Policy

## Supported Versions

| Version | Supported          |
| ------- | ------------------ |
| 2.x.x   | :white_check_mark: |
| 1.7.x   | :white_check_mark: |
| < 1.7.x   | :x: |

## Reporting a Vulnerability

To report a vulnerability, please create an issue on this repo, selecting `Blank Issue` as the template, and then fill it out with all details you have
